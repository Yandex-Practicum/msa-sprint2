🧪 ТЕСТ FEATURE FLAG
====================

Запуск контейнера с включенным флагом:
--------------------------------------
docker run -d --name booking-test \
  -e ENABLE_FEATURE_X=true \
  -e ENVIRONMENT=staging \
  -p 8080:8080 booking-service:latest

Проверка feature endpoint:
--------------------------
curl http://localhost:8080/feature

Результат:
----------
{
  "feature": "X",
  "enabled": true,
  "message": "Feature X is enabled in staging environment!"
}
