# Список изменений
## Бронирования:
- Добавлен вызов получения бронирований пользователя по его id.
- Добавлена проверка пользователя.
- В случае неправомочного запроса возвращается ошибка Access denied.
- Добавлен тип Hotel
- В резолвер Booking добавлен резолв поля hotel по id отеля, который берётся из бронирования

## Файл отеля:
- Добавлена функция getHotelById, запрашивающая по REST данные об отеле по его id
- Функция использована в Query и в резолвере

## Файл gate-way
- Добавлен проброс заголовков запроса в подграфы
