## Внесенные изменения

### gateway
- добавлен проброс заголовка userid в запросах через ApolloGateway

### hotel-subgraph
- добавлен hotel_loader.js выполняющий запросы к monolith с батчингом и кешированием данных
- добавлены резолверы схемы использующие hotel_loader

### booking-subgraph
- добавлен booking_service.js выполняющий gRPC запросы к микросервису бронирования
- добавлена мутация createBooking для создания бронирования
- добавлены резолверы схемы использующие booking_service
- в резолверах интегрирован ACL для ограничения доступа по полю userid

### добавлен promocode-subgraph
- модуль для выполнения REST запросов к monolith
- gql схема с расширением Booking
- query резолверы для получения активных промокодов и валидации
