# Список изменений

## Бронирования
- Добавлен метод получения бронирований пользователя по его `id`.
- Реализована проверка прав доступа: при недостатке полномочий возвращается ошибка **`Access denied`**.
- Введён GraphQL-тип `Hotel`.
- В резолвере типа `Booking` добавлена логика разрешения поля `hotel` — отель загружается по `hotelId`, указанному в бронировании.

## Файл отеля
- Реализована функция `getHotelById(id)`, выполняющая REST-запрос к сервису отелей для получения данных по идентификатору.
- Функция используется как в корневом резолвере `Query`, так и в резолвере поля `hotel` внутри `Booking`.

## Шлюз (Gateway)
- Настроена передача исходных HTTP-заголовков запроса во все подграфы для обеспечения сквозной авторизации и контекста.
