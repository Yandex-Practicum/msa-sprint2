# Пояснение

Каждый запрос bookingsByUser с hotel { ... } вызывает DataLoader.
DataLoader батчит все hotel IDs за один тик event-loop и делает Promise.all параллельных
HTTP-запросов к /api/hotels/{id}, вместо N последовательных запросов.
